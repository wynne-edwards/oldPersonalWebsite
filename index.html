Smartsheets To Google Calendar

About:

My Smartsheets to Google Calendar Script was a program I designed and built for Emerald Consulting.
The program's purpose was to take a large amount of data from one massive database and create an individual calendar for each employee to view their tasks for the upcoming months. Of course, some challenge faced with this was the lack of compatibility between Smartsheets and other APIs. The first of which was getting the data into the Google ecosystem. I solved this by sending a daily CSV file of the Smartsheets file to a google email, where this data now enters the world of Google. From there, it is read and manipulated using google script (javascript), taking this data and sending it to each person's google calendar. The Google Ecosystem is excellent for this due to triggers within google script and their filter function in google sheets.
